# The Concordentary

*A digital concordance and commentary for the Book of Common Prayer and other Anglican texts.*

The Concordentary gathers public-domain prayer book sources, liturgical texts, structured markdown files, and interpretive notes so that prayers, phrases, patterns, and echoes can be studied in relation rather than isolation.

This project is under active construction.
